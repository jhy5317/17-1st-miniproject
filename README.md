# 17-1st-miniproject

## 후속 작업

- 최근 10일간 환율 데이터를 요청할 수 있을 때 통화별 환율 추이 그래프를 추가합니다.

## API 설정

`.env.example`을 복사해 `.env`를 만들고 각 기관에서 발급받은 키를 입력합니다.

```env
AUTH_KEY=한국수출입은행_API_키
ECOS_API_KEY=한국은행_ECOS_API_키
ECOS_CACHE_TTL_SECONDS=3600
```

한국은행 ECOS 연동은 주요국 통화의 대원화환율(`731Y001`, 일별)을 사용합니다.

```text
GET /api/ecos/exchange/USD
```

현재 ECOS 연동 통화는 USD, JPY(100), EUR, CNH, GBP, CAD, CHF, HKD, AUD, NZD입니다.

## 환율 계산기

```text
GET /calculator
```

원화와 외화 간 계산, 외화와 외화 간 교차 계산을 지원합니다. 계산에는 한국수출입은행 매매기준율을 사용하며 100단위 고시 통화는 1단위 가격으로 보정합니다.

## CSV 다운로드

전체 환율 화면의 `CSV 다운로드` 버튼으로 현재 표시 기준일의 환율 데이터를 내려받을 수 있습니다.

```text
GET /rates/download.csv
```

파일명은 `exchange_rates_YYYYMMDD.csv`이며, 엑셀에서 한글이 깨지지 않도록 UTF-8 BOM을 포함합니다. 통화코드, 통화명, 전신환 매입·매도율, 매매기준율과 장부가격을 제공하며 KRW는 제외합니다.

## 최근 10영업일 환율

메인 화면에서 주요 통화 카드를 선택하면 한국은행 ECOS 데이터를 이용한 팝업이 열립니다. 전 영업일 대비 등락, 최근 10영업일 그래프, 최고·최저, 첫날 대비 변동률과 상승·하락 일수를 표시합니다.

## 데이터 최신 상태

메인, 전체 환율, 계산기와 ECOS 그래프에서 실제 데이터 기준일과 확인 시각을 표시합니다. 요청일과 기준일이 같으면 `오늘 데이터`, 다르면 `최근 영업일 데이터`로 안내하며 저장 CSV를 사용한 경우에도 실제 CSV 날짜를 기준일로 표시합니다.

## ECOS 캐시

통화별 최근 10영업일 조회 결과를 `cache_data/ecos`에 저장하고 기본 1시간 동안 재사용합니다. 캐시 유효시간은 `ECOS_CACHE_TTL_SECONDS`로 변경할 수 있습니다. ECOS API 장애 시 마지막 정상 캐시가 있으면 해당 데이터를 표시하며 캐시 폴더는 Git에 포함하지 않습니다.
